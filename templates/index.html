FSND Project 2: Book Catalog App


In this web app, you can catalog and categorize books, either by genre or as uncategorized books. Click on a specific genre or book to view, add, or delete books.

In order to add a book by a specific genre, select the genre for which you are trying to create a book and select 'Add Book'; if you wish to make a book that is not assigned to a specific genre, create the book from the home page.
